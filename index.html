The products

Four products. One knowledge infrastructure.

01Parser

Turn unstructured content into structured evidence.

Converts documents, scans, images, audio and video into structured, machine-readable evidence with provenance.

standalone API · usage-based
View Parser → How provenance works →
02Canonical

Turn evidence into consistent, verifiable knowledge.

Multiple conflicting representations become one governed knowledge object. Every change creates a new version — nothing is ever overwritten.

standalone API · usage-based
View Canonical →
03Brain

AI forgets. Brain doesn’t.

Brain is persistent knowledge that survives conversations, sessions, models and agents — memory that accumulates instead of vanishing.

standalone API · usage-based
View Brain →
04Observatory

Reality changes. Knowledge should too.

Watches what your Brain knows against reality. When reality changes, knowledge updates — through Parser and Canonical, never by editing the Brain.

standalone API · usage-based
View Observatory →
